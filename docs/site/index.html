Cocapn — The npm of AI Agents

An open-source agent runtime by Superinstance. Deploy themed AI instances in 60 seconds.

Get Started View on GitHub

Quick Start

npx create-cocapn my-instance && cd my-instance && cocapn deploy

Your AI instance live on Cloudflare Workers in under a minute.

Features

🔧 Self-Assembly

Agents automatically install skills, tools, and cloud modules based on your project context. Zero configuration required.

🔍 Hybrid Search

Combines semantic vector search with keyword matching. Find anything in your codebase, notes, or documents instantly.

🌳 Tree Search

Explore your knowledge graph with tree-based search. Follow connections between facts, wiki pages, and tasks.

🔌 Plugin System

Extend agents with npm-style plugins. GitHub integration, Slack notifications, data analysis — installable in seconds.

🚀 Fleet Protocol

Multi-agent coordination across devices and domains. Your desktop, laptop, and cloud working as one team.

🎨 Template Marketplace

Themed deployments for every use case: personallog.ai, makerlog.ai, studylog.ai, DMlog.ai, and more.

Why Cocapn?

Cocapn is repo-first and offline-first. Your AI agent's memory lives in Git — private repo for secrets, public repo for published content. The bridge runs locally on your machine. Cloudflare Workers is optional, available for 24/7 background tasks and fleet coordination.

Unlike closed-source AI assistants, Cocapn gives you full control over your data, extensibility through plugins, and freedom to deploy anywhere.
