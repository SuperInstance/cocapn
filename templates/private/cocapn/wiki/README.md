# Wiki

This is the human-readable knowledge base for **{{username}}**'s Cocapn instance.

Agents read and write wiki pages to accumulate shared knowledge. Organise pages
however makes sense for your domain — by project, topic, or person.

## Suggested structure

```
wiki/
├── README.md           # This file — index of the wiki
├── projects/           # One file per active project
├── people/             # Notes on collaborators, contacts
├── tools/              # How-to notes on tools and workflows
└── decisions/          # Architecture decision records (ADRs)
```

Add pages as plain Markdown files. Link between them with relative paths.
